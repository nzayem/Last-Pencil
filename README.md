# Last-Pencil

Last Pencil game to play against a bot.

GamePlay Example:

```
How many pencils would you like to use:
10
Who will be the first (John, Bot):
Bot
||||||||||
Bot's turn!
1
|||||||||
John's turn!
> 2
|||||||
Bot's turn!
2
|||||
John's turn!
> 2
|||
Bot's turn!
2
|
John's turn!
> 1
Bot won!

```
